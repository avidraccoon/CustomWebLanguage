Welcome to my custom MultiScript(A programming language developed for this project) compiler. It is named this because it will be able to be compiled to many different languages.
Currently, it only has basic features and can only be compiled to Javascript

To create a variable called <variable name> do.
var <variable name>
To set the value of a variable do.
<variable name> = <value>
You also can create and assign a variable at the same time
var <variable name> = <value>
